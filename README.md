# TO-DO-LIST

--- 

## English

### About the project
A To-Do List app created to practice working with the DOM, events, rendering data from an array of objects, and fetching data from a local JSON file. The app is available in two languages — English and Slovak.

## Live demo
- GitHub Pages: https://hbibiana.github.io/to-do-list/

## Built with
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (icons)
- Google Fonts — Playfair Display, Caveat, Dancing Script, Pacifico, Roboto Mono

## Features
- Adding tasks via a form
- Marking tasks as completed using checkboxes
- Filtering tasks – all / done / to do
- Editing tasks via edit button
- Deleting tasks
- Completed tasks counter
- Font style selector - loaded from a local JSON file using Fetch API
- Bilingual support (EN / SK) with language saved in localStorage
- Tasks and selected font saved in localStorage

## How to run locally
1. Clone the repository or download it
2. Open `index.html` in your browser

No installation or dependencies required.
