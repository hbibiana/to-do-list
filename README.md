# TO-DO-LIST
Jednoduchá TO DO aplikácia, ktorú som spravila na precvičenie DOM, eventov a renderovania dát z poľa.

## Funkcie
- pridanie úloh cez formulár
- označenie úloh ako dokončených (checkboxy)
- filtrovanie úloh - všetky/dokončené/nedokončené
- vymazanie úlohy
- počítadlo hotových úloh

## Demo 
- GitHub Pages: https://hbibiana.github.io/to-do-list/

## Ako to spustiť lokálne
1. Naklonuj repozitár alebo ho stiahni
2. Otvor `index.html` v prehliadači

## Použité technológie
- HTML
- CSS
- JavaScript (DOM, eventy, práca s poľom objektov)
